# ERP Assistência Técnica e Estoque

Sistema ERP para controle de estoque de peças de celulares, assistência técnica, compras, vendas e ordens de serviço.

## Tecnologias

### Frontend
- Flutter Desktop
- Material Design 3

### Backend
- Node.js
- Express
- Prisma ORM

### Banco de Dados
- PostgreSQL

## Módulos

- Login
- Dashboard
- Produtos
- Estoque
- Clientes
- Fornecedores
- Compras
- Vendas
- Ordens de Serviço
- Relatórios
- Notificações
